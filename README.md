# 🤖 YOLO Object Detection System

## 📌 Overview
This project is a real-time object detection system using YOLOv8 and OpenCV. It supports mobile camera input using DroidCam and includes features like person tracking and phone detection alerts.

---

## 🔥 Features
- Real-time object detection
- Mobile camera integration (DroidCam)
- Person tracking
- Phone detection alert 🚨
- Object counting

---

## 🛠️ Tech Stack
- Python
- OpenCV
- Ultralytics YOLOv8
